# Grok Image Tool

xAI または Venice AI の画像編集 API (どちらも grok imagine image edit) を使って、複数の画像をまとめて一括編集できる Windows 用 GUI ツールです。

## 機能

- フォルダ・ファイルのドラッグ＆ドロップで画像を登録
- プロンプトを入力してまとめて API 送信
- 処理済みファイルの自動スキップ（中断→再開対応）
- 失敗ファイルの自動リトライ
- 保留リストで処理対象を一時退避
- エラーログの CSV 保存

## 動作環境

- Windows 10 / 11
- Python 3.10 以上（ソースから起動する場合）
- または付属の `GrokImageTool.exe`（Python 不要）

## インストール（ソースから起動する場合）

```bash
pip install -r requirements.txt
```

起動：
```bash
python main.py
```
または `起動.bat` をダブルクリック。

## APIキーの取得

### xAI
1. https://console.x.ai/ にアクセスしてアカウントを作成
2. API Keys でキーを発行してコピー

### Venice AI
1. https://venice.ai/ にアクセスしてアカウントを作成
2. API → Generate Key でキーを発行してコピー

APIキーは発行時に1回しか表示されません。忘れずにコピーしてください。<br>
また、どちらも従量課金です。使いすぎにはご注意ください。

## 設定

アプリ起動後、「保留リスト」タブの「設定」欄に API キーを入力して「保存」ボタンを押してください。設定は `settings.json` に保存されます（リポジトリには含まれません）。

## 使い方

1. 「処理対象」タブの登録エリアに画像をドロップ（またはボタンで選択）
2. 出力フォルダを指定
3. プロンプトを入力（例: `背景を白に変えて`）
4. 「開始」ボタンを押す

処理中に「中断」を押すと停止します。次回「開始」を押すと、出力済みのファイルはスキップして続きから再開します。
